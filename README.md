# Number Guessing Game

A beginner-friendly Python command-line game where the user attempts to guess a secret number within a limited number of tries.

This project demonstrates fundamental Python concepts such as:
- Loops
- Conditional statements
- User input handling
- Basic game logic
- Attempt tracking

---

# Features

✔ Fixed secret number  
✔ 6 attempt limit  
✔ Hint system when the guess is close  
✔ Win/Lose message  
✔ Clean and simple CLI interaction  

---

# Technologies Used

- Python 3
- Command Line Interface (CLI)
